# Frontend — Unified Dashboard

**Owner:** Person 3

## Overview
Next.js dashboard integrating all modules into a single UI.

## Tech Stack
- Next.js 14 (App Router)
- React
- TypeScript
- Tailwind CSS

## Setup
```bash
cd frontend
npm install
npm run dev
```

## Directory Guide
- `src/components/dashboard/` — main dashboard views
- `src/components/currency/` — counterfeit detection UI
- `src/components/fraud/` — fraud network visualization
- `src/components/scam-shield/` — scam shield chat UI
- `src/services/` — API client functions
