# ML — Counterfeit Currency Detection

**Owner:** Person 1

## Overview
EfficientNet-based model for detecting counterfeit currency from images.

## Tech Stack
- PyTorch
- EfficientNet
- OpenCV
- Albumentations

## Setup
```bash
cd ml
pip install -r requirements.txt
```

## Directory Guide
- `datasets/` — training data (not committed)
- `models/` — saved model weights (not committed)
- `src/training/` — training scripts
- `src/inference/` — inference pipeline
